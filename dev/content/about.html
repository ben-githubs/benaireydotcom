My Tech Journey

I started getting involved with tech in middle school, when I decided to try my had at game development. The earliest tools I used were simple drag-and-drop interfaces, but I learned a lot about how program organization works and what computer logic looks like. In my later school years, I moved over to Unity 3D, using C# to script more complex game logic while still working within an abstracted game engine.

In university, I started delving into computational physics using MATLAB. Modelling physical systems and simulating different scenarios was very cool, and since physics is a lot of number crunching, I got good at understanding how to structure my code to be as efficient as possible.

MATLAB is incredible proprietary though, so in my free time, I learned how to use Python to replicate MATLAB. Python is much more flexible than MATLAB, and I quickly found myself using it for all sorts of stuff, like a VPN client, auto-filer, and even a small app whcih tracked the contents of all my moving boxes when I swapped appartments. (I didn't lose a single thing that move!)
