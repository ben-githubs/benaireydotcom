
## Detection Engineer

### Panther Labs (July 2024 - July 2025)

As a detection engineer, I got to take ownership of one of the core product experiences at Panther: managing content via CI/CD. My responsibilities were split evenly between managing the out-of-the-box detection content, and developing/maintaining the tools we provide customers for developing their own detections.

**Responsibilities**:

- Develop new, and maintain existing, Panther-managed detection content for customers
- Manage open source github repo, including responding to Issues and reviewing/approving external PRs
- Maintain and develop CLI tooling using Python and Go

**Accomplishments**:

- Migrated release and publish workflows away from `setuptools`
- Created guardrails to more thoroughly test managed content for bugs or noise before releasing broadly to customers
- Streamlined the unit test development process by introducing a log redaction tool
- Co-developed automated red-teaming tools to provide end-to-end detection testing
